## Program for our groups thermal camera drone project (Spring 2024)

### Description of files:
1. **save_fast.py** saves a thermal .tif image from a FLIR Lepton 3.5 thermal camera via a GPIO trigger from a Raspberry PI.

2. **uav_enclosure.step** is the 3D housing for the thermal, visual and NIR sensors ready to be sliced for 3d printing.
